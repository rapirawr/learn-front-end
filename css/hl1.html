Apa itu CSS?

CSS (Cascading Style Sheet) adalah bahasa lembar gaya yang digunakan untuk menentukan presentasi dan gaya dokumen yang ditulis dalam bahasa markup seperti HTML dan XML.

Sejarah CSS

Struktur CSS

Struktur dasar CSS

Struktur dasar CSS terdiri dari tiga komponen utama: selector, properti, dan nilai.

Selector

Selector adalah elemen atau grup elemen HTML yang ingin Anda berikan gaya.

Properti

Properti adalah atribut yang ingin Anda ubah, seperti warna atau ukuran font.

Nilai

Nilai adalah nilai yang diberikan untuk properti tersebut.

Contoh Dasar CSS

Cara memberi warna pada teks


text { 
    color: red;
}

Contoh teks dengan warna merah

Cara memberi warna pada latar belakang


background {
    background-color: red;
}

Contoh teks dengan latar belakang merah

Cara memberi ukuran font


font {
    font-size: 20px;
}

Contoh teks dengan ukuran font 20px

Cara memberi tebal font

   

font {
    font-weight: bold;
}

Contoh teks dengan tebal font

Cara memberi garis bawah pada teks


text {
    text-decoration: underline;
}

Contoh teks dengan garis bawah
