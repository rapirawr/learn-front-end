CSS Learning

Cara menambahkan font di CSS

                
    body {
        font-family: sans-serif;
    }

Ini adalah contoh teks dengan font-family: sans-serif.

Cara menambahkan font dari Google Fonts

                
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;600&display=swap');
body {
    font-family: 'Poppins', sans-serif;
}

Ini adalah contoh teks dengan font-family: 'Poppins', sans-serif.

Cara menambahkan font dari Font Awesome

                
@import url('https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css');
body {
    font-family: 'Font Awesome 5 Free', sans-serif;
}

Ini adalah contoh teks dengan font-family: 'Font Awesome 5 Free', sans-serif.
