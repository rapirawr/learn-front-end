Shorcut untuk HTML

Shortcut

! + Tab: Membuat struktur dasar HTML.
Ctrl + Space: Menampilkan saran otomatis (autocompletion).
Alt + Shift + F: Format kode secara otomatis.
Ctrl + /: Menambahkan atau menghapus komentar.
Ctrl + D: Memilih kata yang sama berikutnya (multiple cursors).
Ctrl + Shift + L: Memilih semua instance dari kata yang sama.
Ctrl + P: Membuka file cepat berdasarkan nama.
Ctrl + Shift + P: Membuka Command Palette.
Ctrl + Shift + Alt + Arrow Down/Up: Menduplikasi baris.
Alt + Arrow Up/Down: Memindahkan baris kode ke atas atau bawah.
Ctrl + K + Ctrl + 0: Melipat semua blok kode (code folding).
Ctrl + K + Ctrl + J: Membuka kembali semua blok kode yang terlipat.
Ctrl + Shift + K: Menghapus baris saat ini.
Ctrl + F: Mencari teks dalam file.
Ctrl + H: Mencari dan mengganti teks dalam file.
Ctrl + B: Menyembunyikan atau menampilkan sidebar di editor.
Ctrl + Shift + O: Melompat ke simbol atau heading di file saat ini.
Ctrl + (backtick): Membuka terminal bawaan editor.
Shift + Alt + F: Memformat file secara keseluruhan.
Ctrl + G: Melompat ke baris tertentu.
Ctrl + Shift + M: Melihat daftar masalah/error di editor.
Alt + Click: Menambah kursor tambahan di lokasi lain (multi-cursor).
